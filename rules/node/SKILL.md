# Node.js Agent Skills

This directory contains specialized rules and guidelines for AI agents working on this Node.js project.

## Available Skills

### 1. Project Structure
- **Path**: `skills/structure/SKILL.md`
- **Description**: Modular Hono project structure using `@apps` alias and feature-based encapsulation.

### 2. Postman Collection Standards
- **Path**: `skills/postman/SKILL.md`
- **Description**: Mandatory rules for generating and updating Postman collections, including automation scripts for token management and CRUD operations.

### 3. Node.js & Hono Development
- **Path**: `skills/node/SKILL.md`
- **Description**: Core rules for Node.js best practices, runtime versioning, package safety, Hono framework patterns, and coding standards.

### 4. Data Transfer Objects (DTO)
- **Path**: `skills/dto/SKILL.md`
- **Description**: Standards for request validation schemas and response transformation patterns.

### 5. Database & Multi-Tenancy
- **Path**: `skills/database/SKILL.md`
- **Description**: Rules for tenant isolation, shared database patterns, and row-level security.

### 6. Object-Relational Mapping (ORM)
- **Path**: `skills/orm/SKILL.md`
- **Description**: Patterns and best practices for Drizzle ORM and TypeORM implementation.

### 7. Cloudflare Platform
- **Path**: `skills/cloudflare/SKILL.md`
- **Description**: Rules for Cloudflare Workers, D1 Database, KV Storage, and R2 Buckets.

### 8. Testing Standards & Patterns
- **Path**: `skills/testing/SKILL.md`
- **Description**: Comprehensive rules for Vitest implementation, Hono/Express testing patterns, database isolation, and test case priority (P0/P1/P2).

### 9. Agent Best Practices
- **Path**: `agent/`
- **Description**: Core rules for [Interaction Protocol](agent/protocol.md), [Data Safety](agent/data_safety.md), documentation standards, [Testing Protocols](agent/testing.md), and versioning protocols.
